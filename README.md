# Android2PC - 局域网跨屏输入神器

Android2PC 是一款极简、高效的局域网跨屏输入工具。它可以将你手机上输入的文字（打字或语音转文字），瞬间发送并粘贴到电脑当前的光标位置。

**核心亮点：**
*   🚀 **极速响应**：基于 UDP 协议，毫秒级传输。
*   🗣️ **智能发送**：支持自动发送模式，专为语音输入优化，无需手动点击。
*   📱 **无缝体验**：手机端输入 -> 自动/手动发送 -> 自动清空 -> 电脑端自动粘贴。
*   🛡️ **纯净安全**：无广告、无多余权限、无需互联网连接（纯局域网）。
*   💻 **零安装**：PC 端仅需一个脚本，即开即用。

---

## 📦 准备工作

### 1. 硬件要求
*   一部 Android 手机（Android 5.0+）。
*   一台 Windows 电脑。
*   **关键点**：手机和电脑必须连接在**同一个局域网**（例如连接同一个 WiFi）。

### 2. 获取软件
在项目文件夹中找到以下两个文件：
*   **Android 端**：`Android_Source\Android2PC.apk`
*   **PC 端**：`run_receiver.ps1`

## 🚀 使用指南

### 第一步：启动 PC 接收端
1.  在电脑上找到 `run_receiver.ps1` 文件。
2.  **右键点击**该文件，选择 **“使用 PowerShell 运行”** (Run with PowerShell)。
3.  等待几秒，窗口会出现绿色的 **"Listening on UDP port 11000"** 提示。
4.  脚本会自动列出你电脑的 IP 地址（通常是 `192.168.x.x` 开头的那个），请**记下这个 IP**。

> **提示**：如果窗口一闪而过，请尝试以管理员身份运行 PowerShell，然后将脚本拖入窗口执行。

### 第二步：配置手机端
1.  将 `Android2PC.apk` 发送到手机并安装。
2.  打开 App，在顶部的 **"PC IP Address"** 输入框中，输入刚才记下的电脑 IP 地址。
3.  App 会自动保存这个 IP，下次打开无需重新输入。
    *   *小技巧*：点击 IP 输入框，可以弹出历史 IP 列表，方便在多台电脑间切换。

### 第三步：开始飞字！
1.  **在电脑上**：把鼠标光标点在你想要输入文字的地方（例如 Word 文档、微信聊天框、浏览器搜索栏）。
2.  **在手机上**：在下方的大输入框里打字，或者使用输入法的语音输入功能。
3.  **发送方式**：
    *   **手动发送**：点击软键盘右下角的“发送”按钮。
    *   **自动发送（推荐）**：使用滑动条设置延迟时间（0.3秒~1.5秒）。当停止输入超过设定时间后，文字会自动发送。
4.  ✨ **见证奇迹**：文字瞬间出现在电脑屏幕上！同时手机输入框自动清空，等待下一条输入。

## ❓ 常见问题 (FAQ)

**Q1: 点击发送后，电脑没反应？**
*   检查手机和电脑是否在同一个 WiFi 下。
*   检查 PC 端脚本是否正在运行（有没有那个黑/蓝色的窗口）。
*   **最可能的原因**：Windows 防火墙拦截了 UDP 11000 端口。请尝试暂时关闭防火墙，或在防火墙设置中允许该端口。

**Q2: 电脑收到了消息（控制台有黄色文字显示），但没有粘贴出来？**
*   请确保你的光标焦点在可输入的文本框内。
*   尝试以**管理员身份**运行 `run_receiver.ps1`。
*   有些安全软件（如企业防泄密软件）可能会禁止脚本操作剪贴板。

**Q3: 手机提示“无法连接”或报错？**
*   这通常是网络不通。请检查 IP 地址是否输入正确（不要输成 127.0.0.1 或 169.254.x.x）。

**Q4: 如何覆盖安装新版本 APK？**
*   直接安装即可。如果提示签名不一致，请先卸载旧版本。

## 🔧 开发者信息

*   **Android 端**：
    *   **通信协议**：UDP (端口 11000)
    *   **核心功能**：`DatagramSocket` + `SeekBar` 延迟控制 + `TextWatcher` 自动发送逻辑
*   **PC 端**：
    *   **脚本**：PowerShell + 内嵌 C# 代码
    *   **API**：Win32 API (`keybd_event`) 实现模拟按键
    *   **环境**：STA Thread 模式运行以支持剪贴板操作
